HTML NOTES

  • html:Hyper text markup language
  • The HTML document itself begins with <html> and ends with </html>.
  • The visible part of the HTML document is between <body> and </body>.
  • An HTML element usually consists of a start tag and end tag, with the content inserted in between:
    • <tagname>Content goes here...<tagname>
    • The HTML element is everything from the start tag to the end tag:
    • <p> My first paragraph.<p>

    • HTML Headings

    • HTML headings are defined with the <h1> to <h6> tags.
    • <h1>defines the most important heading. <h6> defines the least important heading:

    • HTML Horizontal Rules

      • The <hr> tag defines a thematic break in an HTML page, and is most often displayed as a horizontal rule.
      • The <hr> element is used to separate content (or define a change) in an HTML page:

    HTML Paragraphs

    • With HTML, you cannot change the output by adding extra spaces or extra lines in your HTML code.
    • The browser will remove any extra spaces and extra lines when the page is displayed:
    • HTML paragraphs are defined with the <p> tag:
    • HTML links are defined with the <a> tag:

    The HTML <pre> Element

    • The HTML <pre> element defines preformatted text.
    • The text inside a <pre> element is displayed in a fixed-width font (usually Courier), and it preserves both spaces and line breaks

    HTMLAttributes

    • All HTML elements can have attributes
    • Attributes provide additional information about an element
    • Attributes are always specified in the start tag
    • Attributes usually come in name/value pairs like: name="value"

    The href Attribute

    • HTML links are defined with the <a> tag. The link address is specified in the href attribute:
    • href Specifies the URL (web address) for a link

    The src Attribute

    • HTML images are defined with the <img> tag.
    • The filename of the image source is specified in the src attribute:
    • src Specifies the URL (web address) for an image

    The width and height Attributes

    • Images in HTML have a set of size attributes, which specifies the width and height of the image:

    The alt Attribute

    • HTML images are defined with the <img> tag.
    • The source file (src), alternative text (alt), width, and height are provided as attributes:
    • The alt attribute specifies an alternative text to be used, when an image cannot be displayed.
    • alt Specifies an alternative text for an image, when the image cannot be displayed

    The style Attribute

    • The style attribute is used to specify the styling of an element, like color, font, size etc.
    • style Specifies an inline CSS style for an element
    • Setting the style of an HTML element, can be done with the style attribute.
    • HTML style attribute has the following syntax:
        • <tagname style="property:value;">
      • The background-color property defines the background color for an HTML element

        1. <p style="background-color:#ff6347;">#ff6347

          2. #ff6347

        2. < p style="background-color:hsl(9, 100%, 64%);">hsl(9, 100%, 64%)< /p>

          3. hsl(9, 100%, 64%)

        3. <p style="background-color:Tomato;">Lorem ipsum...

          4. Lorem ipsum...

        4. <p style="background-color:rgb(255, 99, 71);">rgb(255, 99, 71)

    • The color property defines the text color for an HTML element:
      • <p style="color:blue;">This is a heading

        • This is a heading

      • <p style="color:red;">This is a paragraph.

        • This is a paragraph.

        • In HTML, a color can be specified by using a color name:

      Border Color

      • You can set the color of borders:
      • <p style="border: 2px solid DodgerBlue;">Hello World

        • Hello World

    • The font-family property defines the font to be used for an HTML element:
      • <h1 style="font-family:verdana;">This is a heading
      • <p style="font-family:courier;">This is a paragraph.

    • The font-size property defines the text size for an HTML element:

        • <p style="font-size:300%;">This is a paragraph</p>

    • The text-align property defines the horizontal text alignment for an HTML element:

        • <p style="text-align:center;">Centered data</p>
