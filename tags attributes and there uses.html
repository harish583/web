    position properties
    static::is not positioned in any special way; it is always positioned according to the normal flow of the page left,right,bottom,top not working
    relative::Setting the top, right, bottom, and left properties of a relatively-positioned element will cause it to be adjusted away from its normal position. Other content will not be adjusted to fit into any gap left by the element.left,right,bottom,top not working

    fixed::means it always stays in the same place even if the page is scrolled. The top, right, bottom, and left properties are used to position the element.float not working,fixed at given top and bottom its not move

    absolute:: positioned element has no positioned ancestors, it uses the document body, and moves along with page scrolling.left,right,bottom,top not working

    sticky::relative until a given offset position is met in the viewport - then it "sticks" in place (like position:fixed). left,right,bottom,top not working from given position to move up to given top(bottom)after that its fixed
